# Pixelit Blooket Private Server

Pixelit is the very first Blooket private server to use pixel art as blooks and badges, created mostly in JavaScript! Created by IzumiiHD, Iamgamedude, Buenar, SOUNDGOD, Prq, ThatPlanet and Packman28. Pixelit is currently still in development, Join our discord server for more updates on the game🐛 https://discord.gg/JMg9uy4Wzk
